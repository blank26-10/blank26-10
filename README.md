## Featured Projects

### Drug Efficacy & Toxicity Prediction
> *Can we predict whether a compound is toxic — before it ever reaches a lab?*

End-to-end ML pipeline for molecular toxicity and drug solubility prediction from SMILES structures. Uses RDKit Morgan fingerprints, physicochemical descriptors, and DeepChem on the Tox21 and ESOL benchmark datasets.

**Stack:** Python · RDKit · DeepChem · XGBoost · Random Forest · scikit-learn · Streamlit  
**Live App:** [drug-toxi-prediction.streamlit.app](https://drug-toxi-prediction.streamlit.app) | **Repo:** [internship_project](https://github.com/blank26-10/internship_project)

---

### Diabetes Risk Prediction
> *Early detection of diabetes using clinical indicators, deployed as a web app.*

ML classification pipeline trained on health survey data. Optimized for recall using a lowered classification threshold to reduce false negatives — because in healthcare, missing a positive case is more costly than a false alarm.

**Stack:** Python · scikit-learn · Random Forest · Streamlit  
**Live App:** [hackathon-c.streamlit.app](https://hackathon-c.streamlit.app)
